# Klasifikasi Diabetes

Proyek ini adalah aplikasi machine learning sederhana untuk klasifikasi diabetes menggunakan data dari file `diabetes_raw.csv`.

## Struktur Proyek

- `app.py` — Script utama untuk training, evaluasi, dan prediksi model klasifikasi diabetes.
- `diabetes_raw.csv` — Dataset mentah berisi data pasien untuk pelatihan model.
- `hasil_klasifikasi_diabetes.csv` — Hasil klasifikasi model pada data uji.
- `prediksi_data_baru.csv` — Hasil prediksi model pada data baru.
- `README.md` — Dokumentasi proyek ini.

## Cara Menjalankan

1. **Pastikan Python sudah terinstall**
2. **Install library yang dibutuhkan**
   
   ```bash
   pip install pandas scikit-learn
   ```
3. **Jalankan aplikasi**
   
   ```bash
   python app.py
   ```

## Penjelasan Singkat

- Model machine learning akan dilatih menggunakan data pada `diabetes_raw.csv`.
- Hasil klasifikasi pada data uji akan disimpan di `hasil_klasifikasi_diabetes.csv`.
- Untuk prediksi data baru, hasilnya akan disimpan di `prediksi_data_baru.csv`.
